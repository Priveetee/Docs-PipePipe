# Copia de Seguridad y Restauración

Todos tus datos en PipePipe (suscripciones, historial, listas de reproducción) se almacenan localmente en tu dispositivo. La app no depende de ningún servicio en la nube. Esto es estupendo para la privacidad, pero significa que tú eres responsable de tus propias copias de seguridad.

## Crear una Copia de Seguridad

Es muy recomendable hacer copias de seguridad de tus datos con regularidad.

1.  Abre el menú principal (icono de hamburguesa en la esquina superior izquierda).
2.  Ve a `Ajustes` > `Copia de seguridad`.
3.  Toca `Exportar base de datos`.
4.  Se abrirá el gestor de archivos de tu teléfono. Navega hasta un lugar seguro para guardar tu archivo de copia de seguridad.

::: tip Buena Práctica
Crea una carpeta dedicada a tus copias de seguridad, por ejemplo, dentro de tu carpeta `Documentos`, llamada `Backup_PipePipe`. Así te resultará fácil encontrar tus copias más adelante.
:::

::: warning
Crea siempre una copia de seguridad antes de actualizar la aplicación. También es un buen hábito crear una cada pocas semanas.
:::
