# Interfaz e idioma

## Interfaz experimental

La interfaz experimental nueva puede cambiar el comportamiento y reinicia PipePipe al cambiarla. Para un problema UI, indica estado, pantalla/pestaña, orientación/tamaño y acción inmediatamente anterior al fallo.

Vuelve a probar una vez sin cambiar la opción. Si comparas deliberadamente activada/desactivada, reinicia tras cada cambio y ofrece dos resultados separados. No mezcles una regresión de interfaz con un fallo de red/extractor solo porque la pantalla estaba visible al ocurrir.

## Idioma, país y localización

Para idioma inesperado, título traducido o error de análisis local, incluye idioma de app, idioma Android, país de contenido, servicio y URL. Un síntoma en zulú, chino u otro idioma puede venir de datos del servicio y no del idioma de interfaz.

Distingue: etiquetas de app en idioma incorrecto; título/descripción devuelto por un servicio; país de contenido sin efecto; error local de fecha/número. Una captura sirve mucho más con estos valores: versión de PipePipe, idioma de app, idioma Android, país de contenido por defecto, servicio y URL.

## Compartir, notificaciones y navegación

Para acción de compartir, control de notificación, botón atrás o solapamiento visual, indica Android, aplicación/pantalla de origen y captura si no revela datos privados. Para abrir un enlace externo, nombra la aplicación origen y acción/intención Android usada: aquí también se aplica la «acción abrir preferida».
