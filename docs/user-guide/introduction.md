# Introduction

PipePipe is an independent Android video client built from the NewPipe lineage.
It supports multiple services and keeps user data on the device rather than
requiring a PipePipe account or cloud service.

## An independent project

PipePipe and NewPipe are separate projects with separate releases, issue trackers,
and implementation choices. A NewPipe workaround, release note, or support answer
may therefore not apply to PipePipe. When reporting a problem, always name the
PipePipe version and the selected extraction endpoint.

## What PipePipe offers

- local subscriptions, history, playlists, and backups;
- background, popup, and music-oriented playback modes;
- per-service extraction, filtering, SponsorBlock, and download features;
- advanced format, subtitle, gesture, and player controls;
- optional YouTube login when a specific need requires it.

## Where to start

Use [Installation](./installation) to choose an update path, then explore the
settings guides. If something fails, use the dedicated [Troubleshooting](/issues/)
section rather than changing unrelated settings at random.
